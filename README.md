## Getting Started

Clone o repositório

```bash
npm install
# ou
yarn
```
depois de instalar as dependencias
```bash
npm run dev
# ou
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.
